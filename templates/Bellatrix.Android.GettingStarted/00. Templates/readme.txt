1. You can create an empty tests project with all required files through dotnet CLI

Install Template
1.1. Open CMD
1.2. Type the below command for the desired template

dotnet new -i Bellatrix.Mobile.Android.GettingStarted
 
Create Project from Template
1.1. Open the folder where you want the files to be placed
1.2. Open CMD
1.3. Type the below command for the desired template

dotnet new Bellatrix.Mobile.Android.GettingStarted
